# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good intent, give constructive feedback, and keep discussions focused on the work.

## Reporting

Report unacceptable behavior by opening a private [security advisory](https://github.com/hitesh-mandloi/claude-sessions/security/advisories/new) on this repository. Reports are visible only to repository maintainers. Maintainers will acknowledge within 7 days.

For non-sensitive matters, open a regular [issue](https://github.com/hitesh-mandloi/claude-sessions/issues).

## Enforcement

The full enforcement guidelines (correction, warning, temporary ban, permanent ban) are defined in the linked Covenant. The maintainer reserves the right to remove comments, commits, code, and contributions that violate this Code of Conduct.
